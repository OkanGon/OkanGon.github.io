Markdown syntax 02

in Markdown syntax

引き続き、HugoにおけるMarkdownの記法について、簡単に説明します。

今回は、見出し／文字修飾／リストなど基本的は記法を説明します。

見出し

#[空白]見出しで「見出し」になります。6段階まで可。

見出し2

##[空白]見出し2

見出し3

###[空白]見出し3

見出し4

###[空白]見出し4

見出し5

#####[空白]見出し5

見出し6

######[空白]見出し6

文字修飾

  • 強調
    記法：**strong text**
    表示：strong text

  • 斜体
    記法：*Italic text*
    表示：Italic text

  • 下線
    記法：<u>Underline text</u>
    表示：Underline text

  • 一重取消線
    記法：~~Deleted text~~
    表示：Deleted text

  • 上付き文字
    記法：上<sup>superscript</sup>
    表示：上superscript

  • 下付文字
    記法：下<sub>subscript</sub>
    表示：下subscript

  • キーボード表示
    記法：<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>
    表示：Ctrl + Shift + A

※ 強調と斜体の*は_でも置き換え可。

リスト

Ordered List (ol)

記法：

1. List Item 1
    1. List Item 1-1
    1. List Item 1-2
        1. List Item 2-1
        1. List Item 2-1
1. List Item 2
1. List Item 3

表示：

  1. List Item 1
    1. List Item 1-1
    2. List Item 1-2
      1. List Item 2-1
      2. List Item 2-1
  2. List Item 2
  3. List Item 3

Unordered List (ul)

記法：

* List Item 1
    * List Item 1-1
    * List Item 1-2
        * List Item 2-1
        * List Item 2-1
* List Item 2
* List Item 3

表示：

  • List Item 1
    • List Item 1-1
    • List Item 1-2
      • List Item 2-1
      • List Item 2-1
  • List Item 2
  • List Item 3

※ 上下に空行が必要です。
※ ulにおいて、*は_でも置き換え可。
※ ネストは空白×4です

表

記法：
1行目：ヘッダ
2行目：セル内のテキスト位置
3行目以降：内容

|Header 1|Header 2|Header 3|
|:--|--:|:--:|
|Division 1|Division 2|Division 3|
|left|right|center|
|left|right|center|
|left|right|center|

表示：

Header 1 Header 2 Header 3
Division 1 Division 2 Division 3
left right center
left right center
left right center

※ 上下に空行が必要です。

引用

記法：

> Quote here.
>
> -- <cite>Benjamin Franklin</cite>

表示：

Quote here.

– Benjamin Franklin

※ 上下に空行が必要です。

略語

ポップアップで非省略語を表示します。

記法：<adbr title="National Basketball Association">NBA</adbr>

表示：NBA

※ NBAの文字上にマウスを移動させてみてください。

水平線

記法： -[空白]-[空白]-

表示：

※ ---でも良いのですが、上の行に文字があると・・・
